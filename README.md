Simple Redux Boilerplate
=====================


### Installation

```
$ git clone https://github.com/mitchbox/simple-redux-boilerplate.git
$ cd your-app-name
$ npm install
```

### Development

```
$ npm start
open http://localhost:3000
```

### Production

```
$ npm run prod
```

### Clean

```
$ npm run clean
```

### Linting

```
$ npm run lint
```

### Testing

```
$ npm run test
```
